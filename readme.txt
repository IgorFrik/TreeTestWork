Тестовое задание: 
Стэк технологий SwiftUI, Combine 
1. Создать структуру дерева, состоящую из узлов (Node) имеющих, название, детей и ссылку на родителя (такие же узлы) 
2. Написать приложение состоящее из одного View с рекурсивной навигацией внутри дерева. (Первый экран открывает корневой уровень (root) и далее мы можем пройти в экраны childs) 
3. Добавить возможность создания и удалений сущностей на каждом уровне 
4. Сохранять состояние дерева на устройстве и подтягивать при следующем входе 
5. Название формировать из случайных 20 байт в виде hex строки по аналогии с адресом кошельков Ethereum
